# PlayFruiter
A Python-based Fruit Machine Game using Python, tKinter, and PyGame.


PlayFruiter is an Fruit Machine Challenge Game in which you must sign in by putting your username, after signing-in you start with a base credit of Â£1 which helps you to play play the game. Each roll costs 20p and player must get all 3 slots of same fruit to win a Jackbot. The Game comes with two Conditions which is if you get 3 same symbols you win a Jackpot, if you get 2 same symbols you win 50p.

The MachRoll is very random and contains different symbols Fish Tablets, Mandarin Orange, Jack Fruit, Aidan Fruit, Cocoa Fruit, Macha Fruit, Jinx Fruit.
